## Descrição projeto nest GitHub

[Nest](https://github.com/nestjs/nest) repositório inicial TypeScript da estrutura.

## Clonar o projeto

```bash
    # Clone o Repositório:
    $ git clone https://github.com/SilvaneiMartins/nest-service-management

    # Entre no Diretório:
    cd nest-service-management
```

## Instalar dependências

```bash
$ yarn install
```

## Executar o projeto

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Executar os test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
